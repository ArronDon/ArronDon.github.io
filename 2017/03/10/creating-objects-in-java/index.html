Creating Objects through Reflection in Java

转自Creating Objects through Reflection in Java with Examples

Class.newInstance()和Constructor.newInstance()的不同

  • Class.newInstance()只能调用无参构造器，Constructor.newInstance()可以调用任何类型的构造器，无论参数有几个。
  • Class.newInstance()需要构造器必须是可见的，Constructor.newInstance()在一定情况下还可以调用私有类型的构造器。
  • Class.newInstance()通过构造器可以抛出任何类型的异常（checked或者unchecked），Constructor.newInstance(）总是抛出封装了这些异常的InvocationTargetExcetion

在Java中创建对象的五种不同方式,转自5 Different ways to create objects in Java with Examples

  1. 使用new关键字
  2. 调用Class.newInstance()方法
  3. 调用Constructor.newInstance()方法
  4. 调用clone()方法
  5. 使用反序列化
    前三种方法都调用了构造器，第四种和第五种没有调用
